# Pet Paradise – Test Plan
**Prepared by:** Gayatri Wakode  
**Date:** 06-Mar-2026

## Introduction
This test plan explains how I will test the Pet Paradise application to make sure login, registration, pets, cart, orders, and logout work correctly.

## Scope
- Login and registration
- Viewing pets
- Adding/removing pets from cart
- Placing orders
- Logout functionality

## Approach
- Run the application locally by executing the source code.
- Test the user interface manually to ensure all features work as expected.
- Validate database entries directly in MySQL for correctness (user data, cart items, orders).
- Log any defects or issues found in GitHub.
