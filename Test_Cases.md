# Pet Paradice Test Cases

## TC_001
Test Scenario: Verify login with valid credentials

Steps:
1. Open Pet Paradice application
2. Enter valid username
3. Enter valid password
4. Click Login button

Expected Result:
User should login successfully and dashboard should open.


## TC_002
Test Scenario: Verify login with invalid credentials

Steps:
1. Open Pet Paradice application
2. Enter invalid username or password
3. Click Login button

Expected Result:
Error message should be displayed.


## TC_003
Test Scenario: Verify pet list display

Steps:
1. Login to application
2. Click View Pets

Expected Result:
List of pets should be displayed.
